1. Выравнивание гистограммы. В методе equalize_hist изображение переводится из BGR в YUV, чтобы не потерять цвет. Затем к третьему каналу применяется equalizeHist и изображение конвертируется обратно в BGR.
2. Локальное выравнивание гистограммы. В методе clahe изображение делится на небольшие части и гистограммы выравниваются отдельно на каждом участке, что позволяет добиться более ровного изображения, без пересвеченных участков.
3. Гауссовское размытие. В методе gaussian_blur указывается ширина и высота ядра Гауссианы, которые должны быть позитивными и нечётными. Также указываются стандартные отклонения по X и Y. Если указано только отклонения по X, то отклонения по Y приравнивается к нему. Если они оба равны нулю, то рассчитываются по размеру ядра.
4. Фильтры Собеля и Лапласа. Для фильтра Собеля можно задать способ поиска производной по X или по Y, поэтому представлено два метода фильтров Собеля (методы sobel_x, sobel_y). В методе laplacian к изображению применяется функция Laplacian и берётся вторая производная по X и Y. Это нужно для обнаружения рёбер.
